# 💼 MERN Stack Portfolio Website

A modern responsive portfolio website built using HTML, CSS, and JavaScript, showcasing my skills, projects, and experience as a MERN Stack Developer.

---

## 🚀 Features

* 📱 Fully responsive design for all devices  
* 🎨 Modern UI with smooth animations  
* 🧭 Sticky navigation bar with smooth scroll  
* 💻 Skills & technical expertise section  
* ⚙️ MERN Stack focused project showcase  
* 📞 Contact section with WhatsApp, Email & Call integration  
* 📄 Downloadable resume option  
* 🔗 Social media links (LinkedIn, GitHub, etc.)  

---

## 🛠️ Tech Stack

* HTML5  
* CSS3  
* JavaScript (ES6+)  
* Font Awesome Icons  
* Google Fonts (Poppins, Ubuntu)  

---

## 📂 Project Structure
```
HTML+CSS+JS Project (Portfolio Website)/
│
├── index.html     # Main homepage - Home, About, Skills, Contact
├── style.css      # All styling - layout, responsive, animations
├── app.js         # JS - navbar, scroll, role animation
│
├── images/        # All portfolio images
│
├── files/         # Resume PDF file
│ └── Resume.pdf
```

## ⚙️ How to Run

Clone the repository:
```
git clone https://github.com/your-username/portfolio-website.git
```
Open project folder:
```
cd portfolio-website
```
Run project:
```
Open index.html in browser
```
